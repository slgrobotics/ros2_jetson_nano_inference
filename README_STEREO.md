## Stereo vision - Jetson Nano, Raspberry Pi 5

Part of this package provides a way to communicate with a *disparity streamer*, running locally on Raspberry Pi 5, or remotely on Raspberry Pi 5 or Jetson Nano.

See https://github.com/slgrobotics/jetson_nano_b01/tree/main/src/stereo for the matching streamer code

Look at https://github.com/slgrobotics/ros2_jetson_nano_inference/blob/main/launch/ros2_disparity_client.launch.py for details.
